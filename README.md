# covid-world-api
covid world stats api built with python flask!

## Built with Python Flask
Clone the repo.
Create a venv for the app.

```sh
pip install -r "requirements.txt"
```
then:

```sh
flask run
```
Open browser and check:
```sh
http://127.0.0.1:5000/
```

### End-Points
/continent or /continent=Asia
/country or /country=India
